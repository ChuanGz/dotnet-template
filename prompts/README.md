# AI Task Prompts

These prompts are for projects generated by `dotnet-service`. They translate the project's selected options and generated conventions into bounded AI tasks; they are not general AI or software-engineering guidance.

Before using a prompt, provide the original `dotnet new` command or recorded option set, the generated project's README, relevant file paths, and its actual validation commands. Replace every placeholder. Do not include secrets or production data.

| Task | Prompt |
|---|---|
| Establish project context | [Use the template](use-template.md) |
| Implement a focused capability | [Add a feature](add-feature.md) |
| Add an HTTP operation | [Add an API endpoint](add-api-endpoint.md) |
| Extend test coverage | [Add tests](add-tests.md) |
| Assess generated changes | [Review generated code](review-generated-code.md) |
| Keep documentation accurate | [Update project docs](update-project-docs.md) |

Use one prompt per task. Reject output that changes selected options, bypasses compatibility rules, or introduces a pattern without a project-specific problem. Review all output and run the generated project's validation commands before committing.
