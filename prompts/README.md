# AI Task Prompts

These prompts help developers make controlled changes to projects generated from this template. Replace every placeholder and provide only the relevant project context. Do not include secrets or production data.

| Task | Prompt |
|---|---|
| Establish project context | [Use the template](use-template.md) |
| Implement a focused capability | [Add a feature](add-feature.md) |
| Add an HTTP operation | [Add an API endpoint](add-api-endpoint.md) |
| Extend test coverage | [Add tests](add-tests.md) |
| Assess generated changes | [Review generated code](review-generated-code.md) |
| Keep documentation accurate | [Update project docs](update-project-docs.md) |

Use one prompt per task. Review all output and run the generated project's validation commands before committing.
